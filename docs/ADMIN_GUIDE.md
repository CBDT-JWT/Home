# 管理员系统使用指南

## 🔐 登录管理后台

访问：**http://www.weitao-jiang.cn/admin_login.html**

默认账户：
- 用户名：`admin`
- 密码：`admin123`

⚠️ **重要**：首次登录后请立即修改密码！

## 📊 管理功能

### 1. 项目管理

登录后可以在管理面板中：
- ✅ 创建新项目
- ✅ 上传缩略图
- ✅ 上传 PDF/Markdown 文件
- ✅ 添加外部链接
- ✅ 编辑现有项目
- ✅ 删除项目
- ✅ 设置项目显示/隐藏
- ✅ 调整项目排序

### 2. 内容类型

支持三种类型的项目：

#### PDF 文档
- 上传 PDF 文件
- 在线预览

#### Markdown 文档
- 上传 .md 文件
- 可在浏览器中打开

#### 外部链接
- 填写任何 URL
- 点击项目卡片直接跳转

### 3. 文件上传

- 支持的图片格式：PNG, JPG, JPEG, GIF
- 支持的文档格式：PDF, MD
- 文件自动保存到 `static/uploads/` 目录
- 文件名自动添加时间戳避免重名

## 🎨 首页展示

项目会自动显示在首页的"我的项目"区域：
- 左侧显示缩略图（或默认图标）
- 右侧显示标题、描述、类型标签
- 点击卡片打开对应内容
- 支持响应式设计（移动端适配）

## 🔧 API 端点

### 管理员认证
- `POST /api/admin/login` - 登录
- `POST /api/admin/logout` - 登出
- `GET /api/admin/check` - 检查登录状态

### 文件上传
- `POST /api/admin/upload` - 上传文件（需要登录）

### 项目管理
- `GET /api/admin/projects` - 获取项目列表
- `POST /api/admin/projects` - 创建项目（需要登录）
- `PUT /api/admin/projects/<id>` - 更新项目（需要登录）
- `DELETE /api/admin/projects/<id>` - 删除项目（需要登录）

## 📝 使用流程

### 创建新项目

1. 登录管理后台
2. 填写项目信息：
   - 项目标题（必填）
   - 项目描述
   - 上传缩略图
3. 选择内容类型：
   - PDF：上传 PDF 文件
   - Markdown：上传 MD 文件
   - Link：填写 URL
4. 设置排序和可见性
5. 点击"保存项目"

### 编辑项目

1. 在项目列表中点击"编辑"
2. 修改项目信息
3. 点击"保存项目"

### 删除项目

1. 在项目列表中点击"删除"
2. 确认删除

## 💡 提示

- 默认情况下，如果没有项目，首页会显示预设的"学习笔记"和"五子棋游戏"
- 创建项目后，首页会动态加载数据库中的项目
- 排序数字越大，项目显示越靠前
- 可以设置项目为"隐藏"，这样只有管理员能看到

## 🔒 安全建议

1. 立即修改默认密码
2. 定期更新密码
3. 不要分享管理员账号
4. 定期备份数据库文件

## 🐛 常见问题

**Q: 忘记密码怎么办？**
A: 需要在服务器上运行 Python 脚本重置密码。

**Q: 上传文件大小限制？**
A: 默认由 Nginx 和 Flask 配置决定，建议单个文件不超过 10MB。

**Q: 如何添加更多管理员？**
A: 目前需要在数据库中手动添加，后续可以开发管理员管理功能。
